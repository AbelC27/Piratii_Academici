# Treasure Map Background

## Instructions

Save the treasure map image you shared as **`treasure_map_bg.jpg`** in this folder.

The image should be the one with:
- Ocean background (turquoise/blue water)
- Large island with sand/desert terrain
- Wooden frame border
- Mountains, trees, and various terrain features
- Compass rose in the corner

Once saved, the pirate map page will display it as the background with checkpoint emojis positioned on top of it!

## File Requirements
- **Filename**: `treasure_map_bg.jpg` (exactly this name)
- **Format**: JPG, JPEG, or PNG
- **Location**: This folder (`backend/app/static/app/images/`)

The CSS is already configured to use this image at:
```css
background-image: url('/static/app/images/treasure_map_bg.jpg');
```
